## Responsive Behavior

### Desktop (>640px)
- 2-column grid for cards
- Full "Bookmarks" text visible
- Full category names in tabs
- Sidebar width: 448px (sm:max-w-md)

### Mobile (<640px)
- Single column grid for cards
- "Bookmarks" text hidden, icon only
- Icon-only tabs
- Sidebar width: 100% (full screen)

## Color Coding

### Categories:
- **AI**: Purple background, purple text
- **Tech**: Blue background, blue text  
- **Web Dev**: Green background, green text

### Badges:
- **Smart Feed**: Outline badge with Sparkles icon
- **Bookmark Count**: Secondary badge with number

### Icons:
- **Not Bookmarked**: Default color (muted)
- **Bookmarked**: Primary color, filled
- **Hover**: Shadow transition
